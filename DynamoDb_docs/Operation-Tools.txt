GUI Console:

The GUI console or the AWS Management Console for Amazon DynamoDB can be found
at the following address − https://console.aws.amazon.com/dynamodb/home

It allows you to perform the following tasks −

CRUD
View Table Items
Perform Table Queries
Set Alarms for Table Capacity Monitoring
View Table Metrics in Real-Time
View Table Alarms
